# TDD Calculator Assignment

## Objective

Implement a calculator using the Test-Driven Development (TDD) approach.

## Red-Green-Refactor Cycle

### Red
- Write a failing test.

### Green
- Implement the minimum code needed to pass the test.

### Refactor
- Improve the code while ensuring all tests continue to pass.

## Run Tests

```bash
python -m unittest test_calculator.py
```
